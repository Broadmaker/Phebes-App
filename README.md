# 📦 POS Application for Phebe's Korean Store

## 📖 Overview

This is a custom-built Point of Sale (POS) application developed using **React Native** and **Expo**.  
It is designed exclusively for **Phebe's Korean Store** to manage sales and product listings in a simple, efficient way.

---

## 🚀 Features

- **Home (Menu)**: Quick access to main app sections.
- **Sales**: Record new sales transactions.
- **Sale History**: View past transactions for tracking and reference.
- **Product Listing**: Manage and view available products.
- **Print Sales**: Generate and print sales records for customers or store documentation.

---

## 📲 Installation

1. Obtain the APK file provided by the developer.
2. Transfer the APK to your Android device.
3. Open the APK file and install the application.
4. Launch the app from your device’s app drawer.
